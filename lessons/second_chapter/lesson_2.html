Введение

Python - высокоуровневый мультипарадигменный интерпритируемый язык программирования. Главной идеей Python является простота и удобство для программиста. Код написанный на Python всегда меньше и проще для чтения, чем код написанный на других языках. Python чаще всего используется для написания серверной части веб-сайтов и веб-приложений, анализа данных и создания скриптов и чат-ботов.

Начало работы

Для написания кода на любом языке программирования всегда требуется 2 вещи - редактор кода в котором мы будем писать код и среда исполнения языка программирования, в нашем случае это интеприратор Python. Разработчики на Python часто используют Visual Studio Code или Pycharm для написания кода. Для новичков рекомендуется использовать Pycharm потому-что, в нем имеется автоматическая подсветка ошибок, а также подскази при написании кода и исправлении ошибок.

Переменные и простые математические операции

Переменная - это зарезервированное пространство в памяти компьютера, которое хранит какое-либо значение и имеет свой индетификатор, который позволяет обращаться к хранимому значению. Для создания переменной в Python необходимо написать название идентификатора переменной, далее написать символ "=" и после указать хранимое в переменной значение. Пример: a = 5
